The Princeton Ethiopian Miracles of Mary digital humanities project (PEMM) analyzes the Gəˁəz miracle stories written about the Virgin Mary in highland Ethiopia. This PEMM Incipit Tool was developed to allows users to catalog miracle stories properly across manuscripts. Type up an incipit (unique opening line) from a story in a Täˀammərä Maryam manuscript, paste it here, and find out which story it is.

Search contains high-confidence incipits from PEMM data. More information about each story is available in project data on Github, or you can also consult a plain list of titles.
